# How to Use the Torus Instructor Dashboard — RISE Rwanda

A self-contained slide deck that walks facilitators through using the OLI Torus instructor dashboard across the teaching cycle: before, during, and after class. Each slide pairs a guiding question with the steps to answer it and the matching dashboard view.

## Viewing

Open `index.html` in any browser. Navigate with the arrow buttons, the dot indicators, or the left and right arrow keys.

## Contents

An intro slide outlining the three phases, followed by seven question slides:

- **Before Class** — How is my class doing? Which students may need support? What concepts are students struggling with?
- **During Class** — What misconceptions should I address today? Where should I spend class time?
- **After Class** — Did students understand the material? What should I improve next time?

Built for RISE Rwanda.
